NOTES

There are several flavours of cubical type theory. One take its roots from BCH paper that was formally given in CCHM and implemented in Haskell as cubicaltt. The second lineage is cartesian cubical type theory thatw was given in ABCFHL and AFH papers and implemented as RedPRL, yacctt, and Arend.

cubicaltt

The cubicaltt is a cubical type checker by Anders Mörtberg that supports path interval, its connections, homogeneous path compositions, higher inductive types, gluening of types and values. Recursive HITs have complete support only in hcomp version.

  • groupoid/cubical — Groupoid Infinity for cubicaltt.
  • groupoid/hcomp — Hopf Fibrations.

Arend

Arend is a cartesian cubical type checker by JetBrains that supports HITs, extensible records, type classes, implicit variables, contains reasoning library.

  • groupoid/arend — Groupoid Infinity for Arend

Lean

Ground Zero is a cubical model and cubical base library embedded into Lean 3 prover.

  • ground/zero — Ground Zero for Lean
